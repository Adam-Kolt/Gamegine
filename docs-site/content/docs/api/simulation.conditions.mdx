## Functions

### RobotInZone

```python
def RobotInZone(robot_name: str, zone: gamegine.representation.bounds.Boundary) -> Callable[[gamegine.simulation.game.GameState], bool]
```

Returns a condition that is True if the robot's center is within the zone's 2D projection.

:param robot_name: Name of the robot to check.
:param zone: The Boundary object representing the zone.

### RobotTouchingObject

```python
def RobotTouchingObject(robot_name: str, object_boundary: gamegine.representation.bounds.Boundary) -> Callable[[gamegine.simulation.game.GameState], bool]
```

Returns a condition that is True if the robot is touching the object (circle-polygon collision).
This is approximate: checks if robot center is within distance `radius` of the object polygon.

### RobotTouchingZoneWithRadius

```python
def RobotTouchingZoneWithRadius(robot_name: str, zone: gamegine.representation.bounds.Boundary, radius: float) -> Callable[[gamegine.simulation.game.GameState], bool]
```

Checks if robot (approximated as circle of radius `radius`) touches the zone.

### RobotsContacting

```python
def RobotsContacting(robot1_name: str, robot2_name: str, robots_config: dict[str, gamegine.representation.robot.Robot]) -> Callable[[gamegine.simulation.game.GameState], bool]
```

Returns a condition that is True if the two robots are touching (distance &lt; sum of radii).

:param robot1_name: Name of first robot.
:param robot2_name: Name of second robot.
:param robots_config: Dictionary mapping robot names to Robot definitions (for radii).
